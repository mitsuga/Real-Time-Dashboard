# 📊 Real-Time Dashboard

## Description
This project is a simple real-time dashboard built using Python and Streamlit. It displays live random values that update automatically to demonstrate real-time data visualization.

## Features
- Live data updates
- Random number generation
- Progress bar
- Simple and interactive dashboard
- Built with Streamlit

## Technologies Used
- Python
- Streamlit

## Project Structure
```
Real-Time-Dashboard/
│── app.py
│── requirements.txt
│── README.md
```

## How to Run

1. Install the required package:
```
pip install -r requirements.txt
```

2. Run the application:
```
streamlit run app.py
```
